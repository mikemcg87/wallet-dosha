## Data Privacy & Ethics

### Critical Considerations

**Privacy First:**
- User owns their data
- Opt-in for each domain
- Granular permissions (can share financial but not health)
- Export/delete at any time

**Security:**
- End-to-end encryption
- No selling of individual data
- Aggregated insights only for research
- GDPR/CCPA compliant

**Transparency:**
- Clear about what data is used
- Explain ML model decisions
- Show confidence scores
- Allow user correction

**Respect:**
- Don't use behavioral insights manipulatively
- No dark patterns
- Help users, don't exploit vulnerabilities
- Partner with practitioners, not replace them
